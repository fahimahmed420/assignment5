Question-2: What is the difference between innerHTML, innerText, and textContent ?
1.innerHTML - Returns the HTML content inside the HTML tag
2.innerText - Returns only the visible text inside an element
3.textContent - Returns all text content inside an element including hidden text.
Question-3: What is event delegation in the DOM?
Event delegation is a technique where you attach an event listener to a parent element instead of individual child elements.
Question-4: What is event bubbling in the DOM?
Event bubbling is a mechanism where an event starts from the target element and moves upwards through its parent elements.Also Use event.stopPropagation() to prevent bubbling
Question-5: How do you create, add, and remove elements using JavaScript?
1.To create an element, we use document.createElement("tagName")
2.To insert the element into the DOM, methods like appendChild() (adds at the end of a parent), prepend() (adds at the beginning), or insertBefore() (places it before another element) are used
3.Calling element.remove() directly removes an element from the DOM, while parent.removeChild(childElement) removes a specific child from its parent.
